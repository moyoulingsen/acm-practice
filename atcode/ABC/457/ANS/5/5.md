**E - 跨越桌布**

时间限制：2.5 秒 / 内存限制：1024 MiB

分数：475 分

**题目描述**

有 N 个单元格排成一行。从左数第 i 个单元格 (1≤i≤N) 称为单元格 i。

现有 M 块布。铺设第 i 块布 (1≤i≤M) 会覆盖从单元格 L_i 到 R_i 的区域。

请回答 Q 个询问。对于第 q 个询问 (1≤q≤Q)，会给出整数 S_q 和 T_q，请解决下面的问题。

判断是否能够从 M 块布中恰好选出两块布，按如下方式铺设：
使得单元格 S_q 到 T_q 被至少一块布覆盖，且除此范围之外的任何单元格都不被任何布覆盖。

**约束条件**

- 1 ≤ N ≤ 2×10^5
- 2 ≤ M ≤ 2×10^5
- 1 ≤ L_i ≤ R_i ≤ N
- 1 ≤ Q ≤ 2×10^5
- 1 ≤ S_q ≤ T_q ≤ N
- 所有输入均为整数

**输入**

输入从标准输入以下列格式给出：

N M
L_1 R_1
L_2 R_2
⋮
L_M R_M
Q
S_1 T_1
S_2 T_2
⋮
S_Q T_Q

**输出**

对于每个询问，输出答案并用换行符分隔。

若可以选出满足条件的两块布，则输出 Yes，否则输出 No。

**样例输入 1**

```
4 3
1 3
1 1
2 4
4
1 4
2 4
1 3
1 1
```

**样例输出 1**

```
Yes
No
Yes
No
```

对于第 1 个询问，选择布 1 和布 3 可以满足条件。

对于第 3 个询问，选择布 1 和布 2 可以满足条件。

对于第 2 个和第 4 个询问，无论选择哪两块布都不能满足条件。

