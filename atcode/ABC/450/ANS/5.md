# E - Fibonacci String
## 原题（英文）
Time Limit: 2 sec / Memory Limit: 1024 MiB
Score : 450 points

### Problem Statement
You are given strings $X$ and $Y$. Define a sequence of strings $S_1,S_2,\dots$ as follows.
$$
\begin{align}
S_1 &= X \\
S_2 &= Y
\end{align}
$$
For $i\ge 3$, $S_i$ is the concatenation of $S_{i-1}$ and $S_{i-2}$ in this order.

For each $i=1,2,\dots,Q$, answer the following problem.

Problem: You are given integers $L_i,R_i$ and a character $C_i$. Find how many times character $C_i$ appears in the $L_i$-th through $R_i$-th characters of $S_{10^{18}}$.

### Constraints
- $X$ and $Y$ are strings of lowercase English letters of length between $1$ and $10^4$, inclusive.
- $1\le Q\le 10^5$
- $1\le L_i \le R_i \le 10^{18}$
- $C_i$ is a lowercase English letter.
- All given numerical values are integers.

### Input
The input is given from Standard Input in the following format:
```
X
Y
Q
L_1 R_1 C_1
L_2 R_2 C_2
⋮
L_Q R_Q C_Q
```

### Output
Output $Q$ lines.
The $i$-th line should contain how many times character $C_i$ appears in the $L_i$-th through $R_i$-th characters of $S_{10^{18}}$.

### Sample Input 1
```
a
b
6
2 7 a
1 3 b
3 7 b
1 9 c
1 1000000000000000000 b
1000000000000000000 1000000000000000000 a
```

### Sample Output 1
```
3
2
3
0
618033988749894848
1
```

### Note
$S_3,S_4,S_5$ are `ba`, `bab`, `babba`, respectively.
$S_{10^{18}}$ is an infinite repetition-like string `babbababbabba…`, and the 2nd through 7th characters contain three occurrences of `a`.

## 中文翻译
### 时间限制
2 秒 / 内存限制：1024 MiB
分值：450 分

### 问题描述
给定两个字符串 $X$ 和 $Y$，按如下规则定义字符串序列 $S_1,S_2,\dots$：
- $S_1 = X$
- $S_2 = Y$
- 对于 $i\ge 3$，$S_i$ 由 $S_{i-1}$ 后拼接 $S_{i-2}$ 得到。

共有 $Q$ 次询问，每次询问给定 $L_i,R_i$ 和字符 $C_i$，求：
在字符串 $S_{10^{18}}$ 中，第 $L_i$ 位到第 $R_i$ 位之间，字符 $C_i$ 一共出现多少次。

### 约束条件
- $X,Y$ 为仅由小写英文字母组成的字符串，长度均在 $1\sim 10^4$ 之间；
- $1\le Q\le 10^5$；
- $1\le L_i \le R_i \le 10^{18}$；
- $C_i$ 为小写英文字母；
- 所有数值均为整数。

### 输入格式
按以下格式从标准输入读入：
```
X
Y
Q
L_1 R_1 C_1
L_2 R_2 C_2
……
L_Q R_Q C_Q
```

### 输出格式
输出 $Q$ 行，第 $i$ 行输出对应询问的答案。

### 样例输入 1
```
a
b
6
2 7 a
1 3 b
3 7 b
1 9 c
1 1000000000000000000 b
1000000000000000000 1000000000000000000 a
```

### 样例输出 1
```
3
2
3
0
618033988749894848
1
```

### 说明
$S_3=\texttt{ba},\ S_4=\texttt{bab},\ S_5=\texttt{babba}$。
$S_{10^{18}}$ 形如 `babbababbabba……`，其中第 2 位到第 7 位一共出现 3 次字符 `a`。
