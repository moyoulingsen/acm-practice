**E - 不平衡的 ABC 子串**

时间限制：2 秒 / 内存限制：1024 MiB

分数：450 分

**题目描述**

给定一个长度为 N 的字符串 S，由字符 A、B、C 三种字符组成。

S 的非空子串共有 \( \frac{N(N+1)}{2} \) 个。请计算其中满足以下条件的子串有多少个：

该子串中 A、B、C 的出现次数互不相同。

即使作为字符串内容相同，只要取自 S 的不同位置，也视为不同的子串并分别计数。

**约束条件**

- \( 1 \leq N \leq 2 \times 10^5 \)
- \( |S| = N \)
- N 为整数。
- S 是仅由字符 A、B、C 组成的字符串。

**输入**

输入由标准输入按以下格式给出：

N
S

**输出**

在一行中输出答案。

**样例输入 1**

```
6
AABBCC
```

**样例输出 1**

```
4
```

从 S 的开头删去 0 个字符、从末尾删去 3 个字符，得到 AAB，满足条件。
从开头删去 1 个字符、从末尾删去 2 个字符，得到 ABB，满足条件。
从开头删去 2 个字符、从末尾删去 1 个字符，得到 BBC，满足条件。
从开头删去 3 个字符、从末尾删去 0 个字符，得到 BCC，满足条件。
除此之外没有其他满足条件的子串。

**样例输入 2**

```
6
ABCABC
```

**样例输出 2**

```
0
```

没有任何子串满足条件。

**样例输入 3**

```
10
ACABCAABAB
```

**样例输出 3**

```
17
```



**E - 不平衡的 ABC 子串 题解**

原出题人：vwxyz

设 \(A, B, C\) 分别表示某个子串中 A, B, C 的出现次数。

令 \(f(X)\) 表示满足条件 \(X\) 的子串个数。

根据容斥原理，所求的答案等于
\[
\frac{N(N+1)}{2} - f(A=B) - f(A=C) - f(B=C) + 3f(A=B=C) - f(A=B=C)
\]
\[
= \frac{N(N+1)}{2} - f(A=B) - f(A=C) - f(B=C) + 2f(A=B=C).
\]

下面介绍如何求 \(f(A=B=C)\) （其他值也可用同样方法求得）。

记 \(A_i, B_i, C_i\) 分别表示 \(S\) 的前 \(i\)（\(0 \le i \le N\)）个字符中 A, B, C 的出现次数。那么从第 \(l+1\) 个到第 \(r\) 个字符的子串中，A, B, C 的出现次数分别为 \(A_r - A_l,\; B_r - B_l,\; C_r - C_l\)。
由于 \(A_r - A_l = B_r - B_l = C_r - C_l\) 等价于 \((A_l - B_l,\; A_l - C_l) = (A_r - B_r,\; A_r - C_r)\)，我们可以对每个 \(i\) 求出 \((A_i - B_i,\; A_i - C_i)\)，并统计有多少对 \((l, r)\) 使得这一对值相等。
用一个关联数组（如 map）来维护 \((A_i - B_i,\; A_i - C_i)\) 的出现频率。
如果某一对值出现了 \(c\) 次，那么从中选择 \(l\) 和 \(r\) 的方案数就有 \(\frac{c(c-1)}{2}\) 种。将它们求和，即可在总计 \(O(N)\) 或 \(O(N\log N)\) 的时间内求出 \(f(A=B=C)\)。

\(f(A=B)\)、\(f(A=C)\)、\(f(B=C)\) 也可以同样求得，由此即可计算出最终答案。
