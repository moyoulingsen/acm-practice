D - 卡片堆查询  
时间限制：2 秒 / 内存限制：1024 MiB  
分数：400 分  

### 题目描述
有 \(N\) 张卡片和 \(N\) 个卡片堆。  
卡片和卡片堆的编号均为 \(1, 2, \dots, N\)。初始时，第 \(i\) 个堆中只有卡片 \(i\)。  

按 \(i = 1, 2, \dots, Q\) 的顺序依次执行以下操作：  

将卡片 \(C_i\) 以及叠放在卡片 \(C_i\) 上方的所有卡片（保持原有顺序）一并移动到卡片 \(P_i\) 的上方。数据保证，在执行操作的前一刻，卡片 \(C_i\) 与卡片 \(P_i\) 处于不同的堆中，且卡片 \(P_i\) 位于某个堆的顶端。  

在所有操作完成后，求出每个堆中的卡片数量。  

### 约束条件
- \(1 \le N, Q \le 3 \times 10^5\)
- \(1 \le C_i, P_i \le N\)
- 按顺序执行操作时，在每次操作的前一刻，卡片 \(C_i\) 与卡片 \(P_i\) 处于不同的堆中。
- 按顺序执行操作时，在每次操作的前一刻，卡片 \(P_i\) 位于某个堆的顶端。
- 所有输入值均为整数。

### 输入
输入以以下格式从标准输入给出：  
```
N Q
C1 P1
C2 P2
⋮
CQ PQ
```

### 输出
设最终第 \(i\) 个堆中的卡片数为 \(A_i\)。按顺序输出 \(A_1, A_2, \dots, A_N\)，以空格分隔。

### 样例输入 1
```
5 4
1 3
4 5
1 4
4 2
```
### 样例输出 1
```
0 3 1 0 1
```
