# D - Make Target 2
## 原题（英文）
Time Limit: 2 sec / Memory Limit: 1024 MiB
Score : 425 points

### Problem Statement
There is a two-dimensional coordinate plane. A lattice point with coordinates $(x,y)$ is painted black if $\max(|x|,|y|)$ is even, and white if it is odd.

Among the pairs of integers $(x,y)$ satisfying $L\le x\le R$ and $D\le y\le U$, find the number of those for which the point $(x,y)$ is painted black.

### Constraints
- $-10^6 \le L\le R\le 10^6$
- $-10^6 \le D\le U\le 10^6$
- All input values are integers.

### Input
The input is given from Standard Input in the following format:
```
L R D U
```

### Output
Output the answer.

### Sample Input 1
```
-4 3 1 3
```
### Sample Output 1
```
10
```

### Sample Input 2
```
-14 14 -14 14
```
### Sample Output 2
```
449
```

### 问题描述
在二维平面直角坐标系中，对于格点坐标 $(x,y)$：
- 如果 $\max(|x|,|y|)$ 的值为**偶数**，则该点被涂成**黑色**；
- 如果 $\max(|x|,|y|)$ 的值为**奇数**，则该点被涂成**白色**。

给定范围 $L\le x\le R$ 且 $D\le y\le U$，请统计这个范围内满足条件的整数坐标对 $(x,y)$ 中，**黑色点**的数量。

### 约束条件
- $-10^6 \le L\le R\le 10^6$
- $-10^6 \le D\le U\le 10^6$
- 所有输入值均为整数。

### 输入格式
输入从标准输入按以下格式读取：
```
L R D U
```

### 输出格式
输出一行，表示答案。

-4 3 1 3
